# Introduction
Learning how to code, that's probably why you're here. In order to become great at coding you must first realize what coding is. Coding is an expression of logic interpreted from your text and read as instructions to manipulate data. Writing code is the expression but in order to express things we need to go through the real art that makes not only coders great, but people too. Learning to code is not merely a path to knowing how to create games learning to code is the journey of becoming a master SELF SUFFICIENT problem solver. In the teachings in this resource self sufficiency and learning how to think differently in order to solve your own problems is what we are really about, nothing less would make you the coder or developer you want to be.
